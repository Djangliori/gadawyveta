# გადაწყვეტა

დროის აღრიცხვისა და დავალებების მართვის აპლიკაცია, აგებული Next.js-ზე.

## ფუნქციონალი

- დროის აღრიცხვა (ხელით შეყვანა)
- დავალებების დამატება, რედაქტირება, წაშლა
- კალენდარი თვის ხედით
- ცხრილი 5 სვეტით:
  - თარიღი
  - დავალების სახელი
  - დახარჯული დრო
  - აღწერა
  - კატეგორია

## ტექნოლოგიები

- **Next.js 15** - React ფრეიმვორკი
- **TypeScript** - ტიპების უსაფრთხოება
- **Tailwind CSS** - მინიმალისტური დიზაინი
- **Vercel Postgres** - მონაცემთა ბაზა

## დაყენება ლოკალურად

1. დააკლონეთ პროექტი
2. დააყენეთ დამოკიდებულებები:
   ```bash
   npm install
   ```

3. შექმენით `.env.local` ფაილი და დაამატეთ Vercel Postgres-ის credentials:
   ```
   POSTGRES_URL="your_postgres_url"
   ```

4. გაუშვით development სერვერი:
   ```bash
   npm run dev
   ```

5. გახსენით ბრაუზერში: http://localhost:3000

## Vercel-ზე Deploy

1. გადაიტანეთ პროექტი GitHub-ზე
2. დააკავშირეთ Vercel-თან
3. დაამატეთ Vercel Postgres ბაზა:
   - Vercel Dashboard → Storage → Create Database → Postgres
4. ავტომატურად დაემატება საჭირო environment variables
5. Deploy!
